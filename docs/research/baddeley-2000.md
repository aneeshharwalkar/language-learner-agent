# Baddeley (2000) — The Episodic Buffer

**Citation (APA):**
Baddeley, A. (2000). The episodic buffer: A new component of working memory? *Trends in Cognitive Sciences*, *4*(11), 417–423. https://doi.org/10.1016/S1364-6613(00)01538-2

---

## Summary

- Introduces the **Episodic Buffer** as a fourth component of the working memory model, supplementing the Phonological Loop, Visuospatial Sketchpad, and Central Executive.
- The buffer is a limited-capacity, multimodal store that **binds information from the subsidiary systems and from long-term memory into a unitary episodic representation** — the mechanism by which new input is integrated with prior knowledge and with context.
- Retrieval from the buffer is assumed to be under conscious attentional control, which makes it the working-memory substrate where situational, narrative, and relational context become available to learning.
- Reframes working memory away from isolated subsystem storage and toward *integration* across verbal, visual, and episodic codes — directly relevant to why decontextualized drill (Phonological Loop only) fails to support conversational use (requires Episodic Buffer binding).
- Provides the canonical citation for the HFACS analysis's Level 2 claim that Duolingo engages only the Phonological Loop and forecloses Episodic Buffer engagement; this is a structural constraint on transfer, not a user motivation issue.

**Key Quote:**
> "A limited capacity system that provides temporary storage of information held in a multimodal code, capable of binding information from the subsidiary systems, and from long-term memory, into a unitary episodic representation."

---

**Rubric Phase:** Phase 4 — supplies the cognitive-architecture foundation for the Level 2 claim that contextual, multimodal encoding is required for conversational transfer and that gamified app interfaces cannot provide it.
